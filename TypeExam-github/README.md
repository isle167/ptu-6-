# ⌨️ TypeExam

> 정보처리기사 실기 · 토익 파트별 타이핑 훈련 앱

단어를 **타이핑하며** 외우는 웹 기반 암기 트레이너입니다.  
별도 설치 없이 HTML 파일 하나로 바로 실행됩니다.

---

## 🚀 실행 방법

### 바로 실행
`index.html` 파일을 브라우저에서 열면 됩니다.

```
index.html 더블클릭 → Chrome / Safari / Firefox 로 열기
```

> ⚠️ IntelliJ 내장 서버(localhost)로 열면 404가 뜰 수 있습니다.  
> 반드시 **파일 직접 열기(`file://`)** 또는 **브라우저에 드래그**해서 실행하세요.

### GitHub Pages로 배포
1. 이 저장소를 Fork 또는 본인 계정에 Push
2. Settings → Pages → Branch: `main`, Folder: `/ (root)` 선택
3. 저장하면 `https://{username}.github.io/{repo-name}/` 으로 접속 가능

## ✨ 기능

| 기능 | 설명 |
|------|------|
| 📚 정보처리기사 실기 | DB · 알고리즘 · 네트워크 · 보안 · SW공학 (파트별 40문항) |
| 📝 TOEIC | Part 5 어법/어휘 · Part 6 문맥 · Part 7 독해 · 비즈니스 영어 |
| 🔀 한/영 혼합 출제 | 영어만 · 한글만 · 랜덤 혼합 선택 가능 |
| ⏱️ 타임어택 | 60 / 90 / 120초 제한 모드 |
| 💡 힌트 | 정답 + 설명 표시, 힌트 사용 시 △ 처리 |
| 📋 오답노트 | 오답 자동 저장, 파트별 분류, 플래시카드(가리기) |
| ✏️ 문제 추가 | 나만의 단어장 추가 및 게임 실행 |
| 🌙 다크/라이트 테마 | 시스템 테마 자동 감지, 수동 토글, localStorage 저장 |
| 🔊 사운드 효과 | 타이핑 · 정답 · 결과 효과음 (음소거 지원) |

## 🎮 게임 방법

1. **정보처리기사 실기** 또는 **TOEIC** 선택
2. 파트 선택
3. **출제 방식** (영어만 / 한글만 / 혼합) 선택
4. **시간 모드** (제한 없음 / 타임어택) 선택
5. 화면에 표시된 뜻을 보고 단어를 타이핑 후 **Enter**
6. 모르면 **💡 힌트** 버튼 → 정답 확인 후 입력 (△ 처리)
7. 결과 확인 후 오답노트 저장

---

## 💾 데이터 저장

오답노트와 커스텀 문제는 브라우저 `localStorage`에 저장됩니다.  
같은 브라우저에서는 새로고침해도 유지됩니다.

---

## 🛠 기술 스택

- **Vanilla HTML / CSS / JavaScript** — 프레임워크 없음
- **localStorage** — 오답노트, 커스텀 단어, 테마 설정 저장
- **CSS Custom Properties** — 다크/라이트 테마 전환
